# Phase 14 ALEA-R example scripts

Copy these files into:

```text
package/ALEA/examples/
```

Expected data files:

```text
package/ALEA/examples/data/paraopeba_annual_max_flow.csv
package/ALEA/examples/data/paraopeba_annual_max_rainfall.csv
package/ALEA/examples/data/paraopeba_annual_mean_flow.csv
```

Run each script from the ALEA package root directory, for example:

```r
setwd("C:/Users/wilso/OneDrive/Projects/R_ALEA/package/ALEA")
source("examples/01_single_site_basic_workflow.R")
```

The scripts are designed to work both with `source()` and when run interactively in RStudio.

All comments and teaching text are in English, and all workflows use only the current ALEA-R public API.
